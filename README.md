# ①課題名
野球カウント登録・表示システム

## ②課題内容（どんな作品か）
- 野球の審判が使用するインジケータ（Ball/Strike/Outカウント）をデジタル化
- 入力（index.php）:スマホ等の端末からカウントを送信
- 登録（write.php）:送信されたデータを時刻と合わせてCSVファイルに蓄積
- 表示（read.php）:履歴を表示
